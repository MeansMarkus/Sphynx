# Sphynx Watch Configurator

Phase 1 foundation MVP for a premium watch configurator. The experience focuses
on a cinematic dark UI, live layered preview, and curated preset builds.

## Highlights

- Luxury landing hero with clear call to action
- Live watch configurator using layered PNG assets
- Five preset builds (dress, dive, tactical, vintage, sport)
- Responsive layout optimized for desktop, tablet, and mobile

## Tech Stack

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- Framer Motion

## Getting Started

Install dependencies and run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to view the configurator.
